# Chevalet Anon Bot

A simple yet full-of-feature telegram bot to send messages to other users anonymously

### Features

- multiple links: users can have multiple anon links and will be
 notified what is the source of each new message thread (which link was used)
- changing the preview name
- general unblocking to unblock everyone you have blocked
- individual unblocking using "unblock me" command
- access for admin to ban/unban/change cid limit/send mass message and so on
